## DESCRIPCIÓN
Este repositorio contiene el desarrollo completo de la Tarea S2 de la asignatura
Programación Orientada a Objetos, aplicando los conceptos de clases, objetos,
constructores y encapsulamiento.
---
## ACTIVIDADES DESARROLLADAS
### Actividad 1: Investigación
Se investigaron las convenciones de Java para clases, atributos, métodos y objetos,
incluyendo ejemplos prácticos para una mejor comprensión.
---

### Actividad 2: Práctica – Clase Libro
Se creó la clase `Libro` con atributos, constructor y método para mostrar información.
Se instanciaron dos objetos y se visualizaron los datos en consola.
---

### Actividad 3: Tipos de Constructores
Se desarrolló la clase `Producto` implementando:
- Constructor vacío
- Constructor con valores fijos
- Constructor con parámetros  
Se crearon objetos para cada tipo de constructor.
---
### Actividad 4: Simulación Cuenta Bancaria
Se implementó la clase `CuentaBancaria` aplicando encapsulamiento con atributos privados,
constructor público y métodos para depositar, retirar y mostrar información.

---

###  Tema: Encapsulamiento
Se explicaron los modificadores de acceso, getters, setters, validaciones y ventajas del
encapsulamiento, aplicando ejercicios prácticos.
